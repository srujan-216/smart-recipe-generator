   ```
   Status: ✅ Running
   ```

   ```bash
   curl http://localhost:5000/api/health
   ```

3. **Generate a recipe**
   - Register → Set preferences → Input ingredients → Click Generate

### For Production Deployment

**See full guide**: `PRODUCTION_DEPLOYMENT.md`

**Quick deploy**:
```bash
# Option 1: Docker
docker build -t smart-recipe .
docker run -d -p 80:5000 \
  -e MONGO_URI=your_uri \
  -e GROQ_API_KEY=your_key \
  -e JWT_SECRET=random_secret \
  smart-recipe

# Option 2: Traditional
npm install
npm start

# Option 3: GitHub Actions (Automated)
git push → GitHub Actions builds & deploys
```
